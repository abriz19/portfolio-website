## Overview

The **OmniFood** project is a responsive restaurant website that presents the menu items, their descriptions, ingredients, and prices. Built using **HTML5**, **CSS3**, and **JavaScript**, the website was designed with simplicity in mind, while providing users with a clear and easy-to-navigate interface. It also integrates a dynamic menu system to handle updates to the food items and prices.

## Key Features

- **Dynamic Menu**: The website dynamically displays dishes with descriptions, ingredients, and prices.
- **Responsive Design**: Built to ensure that users on any device (desktop, tablet, mobile) have a smooth experience.
- **Dish Details**: Clicking on a dish shows more detailed information such as its ingredients and nutritional information.
- **Search Functionality**: Users can search for specific dishes by name, ingredients, or type.
- **Interactive UI**: Interactive elements such as hover effects and clickable details add a modern touch to the user experience.

## Technologies Used

- **HTML5**: For structuring the content and ensuring semantic markup.
- **CSS3**: For styling the website, ensuring a clean and modern look.
- **JavaScript**: For adding interactivity, such as filtering dishes and displaying detailed views.
- **Bootstrap**: For responsive design components and layout.

## Challenges and Learnings

The main challenge was ensuring that the website was optimized for all screen sizes while maintaining a consistent look and feel across devices. Also, managing the dynamic display of dishes and ensuring smooth interactivity required careful planning.

## Outcome

The **OmniFood** project provided a clean and intuitive way for restaurant customers to view the menu and get detailed information about each dish. The website’s responsive design ensures that it delivers a great experience for users, whether they’re on mobile or desktop.
